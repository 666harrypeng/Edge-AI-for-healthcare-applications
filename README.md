# Connection
Connection for Temp/humid (AHT10), Pressure (BMP180), Blood Oxygen (MAX30102):

GND -> GND -> GND

SCK(SCL) -> 22 -> SCL

SDA -> 21 -> SDA

VCC -> 3.3V -> VDD

Connection for Analog Microphone:

GND -> GND -> GND

VCC -> 5V -> VDD

Analog -> 34 -> AO
